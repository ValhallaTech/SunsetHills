The Algorithm

Explore the Sunset Hills solution with detailed code explanation

Algorithm Approach

The most efficient solution uses a right-to-left scan with a running maximum. As we iterate from right to left (east to west), we track the tallest building seen so far. A building can see the sunset if it's taller than all buildings to its right.

Complexity Analysis

Time Complexity:
O(n) Single pass through array
Space Complexity:
O(1) Constant extra space

Step-by-Step Process

  1. Start from the rightmost building (it always sees the sunset)
  2. Track the maximum height encountered so far
  3. For each building moving left: if it's taller than the max, it can see the sunset
  4. Update the maximum height after each comparison
  5. Return the indices of buildings with sunset view

Example Walkthrough

Input: [71, 55, 50, 65, 95, 68, 28, 34, 14]

Index Height Max So Far Can See Sunset?
8 14 0 Yes (14 > 0)
7 34 14 Yes (34 > 14)
6 28 34 No (28 ≤ 34)
5 68 34 Yes (68 > 34)
4 95 68 Yes (95 > 68)
3 65 95 No (65 ≤ 95)
2 50 95 No (50 ≤ 95)
1 55 95 No (55 ≤ 95)
0 71 95 No (71 ≤ 95)

Output: [0, 4, 5, 7, 8] (Buildings 1, 5, 6, 8, and 9 can see the sunset)
