The Algorithm

Explore the Sunset Hills solution with detailed code explanation

Approach

The most efficient solution uses a right-to-left scan with a running maximum. As we iterate from right to left (east to west), we track the tallest building seen so far. A building can see the sunset if it's taller than all buildings to its right.

Complexity Analysis

  • Time Complexity: O(n) - Single pass through the array
  • Space Complexity: O(1) - Only storing the max height and result indices
