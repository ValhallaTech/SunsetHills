Buildings with Sunset View

Adjust the buildings to see which ones can see the sunset...

How to use:

  • Click and drag any building up or down to change its height
  • Buildings highlighted in gold/orange can see the sunset
  • A building can see the sunset if all buildings to its right are shorter
  • Use the Reset button to restore default heights
  • Use the Randomize button to generate random building heights
