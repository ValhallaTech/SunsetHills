Buildings with Sunset View

Adjust the buildings to see which ones can see the sunset...

How to use:

  • Buildings are arranged west to east (left to right)
  • The sunset is to the west (left side of the chart)
  • Buildings highlighted in orange can see the sunset
  • A building can see the sunset if all buildings to its left are shorter
  • Use the Reset button to restore default heights
  • Use the Randomize button to generate random building heights
  • Try different preset scenarios from the dropdown
  • Enter custom heights (comma-separated, 10-100) for manual testing
